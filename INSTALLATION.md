# Pixel Streaming with Nextjs

## installation

```bash
# 1. clone repo
git clone https://github.com/markolofsen/metaeditor.git

# or clone dev version
git clone -b dev https://github.com/markolofsen/metaeditor.git

# 2. Installation
cd ./metaeditor
yarn && yarn dev

# 3. open http://localhost:3000
```
