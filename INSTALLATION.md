# Scalable Pixel Streaming Frontend

## installation

```bash
# 1. clone repo
git clone https://github.com/markolofsen/metaeditor.git

# or clone dev version
git clone -b dev-v2 https://github.com/markolofsen/metaeditor.git

# 2. npm module
cd ./package
yarn && yarn start

# 4. run project
cd ./project
yarn
yarn symlink
yarn dev

# 5. open http://localhost:3000
```
